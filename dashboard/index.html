Firewall Settings

Manage network routing, interface configurations, and peer access.

Backend Mechanism
Activating Client Isolation runs an IPTables/NFTables ruleset on the VPS: iptables -I FORWARD -i wg0 -o wg0 -j REJECT This prevents any device in the subnet from scanning, pinging, or forwarding data packets to another subnet peer.
